# VISION

## Долгосрочное видение

AI Memory OS должен стать непрерывной внешней памятью человека.

Через несколько лет пользователь не должен думать, где лежит информация. Он должен просто спросить:

- что происходило вокруг этой встречи;
- что я обещал;
- какие материалы были по этому проекту;
- почему мы приняли это решение;
- что я думал об этой идее месяц назад;
- какие документы и ссылки относятся к этой поездке.

## Не устройство, а память

Telegram, Web, Mobile, Voice Pendant, Smart Watch и API — это только точки входа.

Настоящий продукт — единая память под ними.

Пользователь может отправить мысль голосом с медальона, документ через Telegram, ссылку из браузера, заметку с телефона, а потом найти все это как часть одного жизненного контекста.

## Каким должен стать продукт

AI Memory OS должен уметь:

- помнить события жизни;
- связывать записи между собой;
- понимать проекты и темы;
- помнить людей и встречи;
- находить старый контекст;
- показывать открытые петли;
- сохранять решения и причины;
- помогать человеку видеть ход собственной жизни и работы.

## Северная звезда

Пользователь должен чувствовать:

> Я больше не теряю нить собственной жизни.

## Чем это отличается от AI-диктофона

AI-диктофон хранит записи.

AI Memory OS хранит контекст.

Запись важна не сама по себе, а потому что она относится к проекту, человеку, встрече, поездке, обещанию, идее или решению.
